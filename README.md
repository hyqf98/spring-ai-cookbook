## 技术栈

- Spring Boot 3.5.8
- Spring AI 1.1.0
- Java 25

## 快速开始

1. 克隆项目
2. 进入对应的子模块目录
3. 查看子模块的 README.md 了解详细使用说明
4. 运行 `mvn spring-boot:run` 启动应用
